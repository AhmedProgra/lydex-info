Technologies utilisées

HTML, CSS et JavaScript (Web Languages)
Bien entendu, nous avons utilisé les trois langages du web: HTML, CSS et JavaScript, car ce sont les principaux langages pour construire tout site web.
Bootstrap (version=5.3.3)
Nous avons utilisé le framework Bootstrap, car il nous fait gagner beaucoup de temps lors du processus de développement.
Font Awesome (version=6.5.1)
Nous avons utilisé Font Awesome, car il fournit des icônes très importantes pour nous dans le project et est également facile à formater et à gérer.
jQuery (version=3.7.1)
Et Bien Sûr, nous avons utilisé le fameux jQuery, qui a considérablement raccourci les processus nécessitant l'utilisation de JavaScript.
Splide JS (version=4.1.2)
Nous avons utilisé Splide JS pour créer une belle galerie d'images pour accompagner chaque article du journal.
Video JS (version=8.16.1)
Nous avons utilisé Video JS pour afficher des vidéos accompagnant des articles ou des interviews sous forme vidéo en utilisant le thème fantastique.
Visual Studio Code (Code Editor)
Nous avons utilisé l'éditeur Visual Studio Code, qui est open source et gratuit, et regorge de plugins pour aider dans le processus de développement.
Git et Github
Nous avons également utilisé Git pour contrôler les versions du site web, et nous l'avons également téléchargé sur GitHub pour conserver les copies du project et la possibilité de modification unifiée.
